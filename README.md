## Running locally

You need Ruby and gem before starting, then:

```bash
# install bundler
gem install bundler

# clone the project
git clone https://github.com/usmanaura47/viral-tool-site.git
cd viral-tool-site

# run jekyll with dependencies
bundle exec jekyll serve
```
